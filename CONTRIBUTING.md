# Contributing Guidelines

Thank you for contributing!

## Steps to Contribute
1. Fork the repository.
2. Create a branch:
   git checkout -b feature-name
3. Make changes.
4. Commit:
   git commit -m "Your message"
5. Push:
   git push origin feature-name
6. Create a Pull Request.

## Code Style
- Write clean code
- Add comments when needed

Thank you for contributing!
